# psam_website
Code der ps-am Webseite

Die Webseite wird auf mehreren Services verteilt:
1. Hostinger wurde benutzt, um ein Domain und eine Email zu kaufen/mieten
2. Github wurde benutzt, um die Dateien und Code der Webseite zu speichern in diesem privaten Repo
3. Netlify ist der main Hosting-Service, es wurde mit Github verknüpft, um die Daten der Webseite zu ziehen und zu hosten
